웹 디자인 원칙과 전략

1. 사용자 중심 디자인

사용자의 니즈와 행동 패턴을 이해하고, 그에 맞춰 디자인을 구성합니다.

2. 간결하고 명료한 레이아웃

복잡하지 않고 명료한 레이아웃으로 사용자 경험을 향상시킵니다.

3. 일관성 있는 브랜딩

일관된 브랜드 아이덴티티를 반영하여 브랜드 인지도를 높입니다.

4. 접근성과 반응형 디자인

다양한 기기와 환경에서 접근 가능하고 반응형 웹 디자인을 적용합니다.

5. 트렌드와 혁신 추구

최신 디자인 트렌드와 기술을 반영하여 혁신적인 디자인을 제공합니다.

프로필

이관형 / 공업디자인학과

컨퍼런스 인턴, 초기 스타트업 디자이너 등 활동, 특이한 분야에 디자인 교집합 만드는 것을 좋아합니다.

잘하는 것

  • (본인이 경험했거나 익숙한 필드의) 사용자 경험을 개선하고, 디자인 씽킹 베이스로 부가가치를 더하는 것
  • 카카오톡 선물하기 문화의 단점과 개선점이 무엇인가? / 한국의 정치 양극화를 해소하기 위해 어떤 디자인적 접근이 필요한가? / 외교 행사에 K팝 댄스를 추가하면 어떤 가치가 나오나?

좋아하는 것

  • (디자인 교집합 발생 가능 영역의) 새로운 사람들과 대화하는 것, 뉴스 보는 것

세상이 필요로 하는 것

  • 문화적 관점의 다양성/ 개방성

10년 후의 Vision

최근의 세상은 점점 개인화되고, 양극화 중이지만, 10년을 거쳐 그것이 극도로 진행돼 갈수록 점점 연결된 문화와 메세지를 향한 고민 역시 많아질 것이라고 생각합니다. 저는 다양한 분야에서 프로젝트 기획 능력을 바탕으로 사람 간의 통합과 이해를 돕기 위해 활동하고 싶습니다.

"우수한 웹 디자인은 사용자 경험과 브랜드 가치를 극대화합니다."
